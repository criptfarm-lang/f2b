# 🤖 F2B PRO Telegram Bot

Ассистент отдела продаж: задачи, фото, прайсы, дебиторка, ответы на вопросы.

---

## 🚀 Запуск за 15 минут (Railway)

### Шаг 1 — Создай бота в Telegram

1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай название: `F2B PRO Assistant`
4. Придумай username: `f2bpro_assistant_bot` (должен заканчиваться на `bot`)
5. **Сохрани токен** — выглядит так: `7234567890:AAFxxx...`

---

### Шаг 2 — Получи API ключ Anthropic

1. Зайди на **console.anthropic.com**
2. Зарегистрируйся / войди
3. API Keys → Create Key
4. **Сохрани ключ**: `sk-ant-...`

---

### Шаг 3 — Залей код на GitHub

1. Зайди на **github.com** → New Repository → назови `f2b-bot`
2. Загрузи все файлы (`bot.py`, `database.py`, `claude_ai.py`, `scheduler.py`, `requirements.txt`)
3. Сделай репозиторий **Public**

---

### Шаг 4 — Задеплой на Railway

1. Зайди на **railway.app** → Login with GitHub
2. New Project → Deploy from GitHub repo → выбери `f2b-bot`
3. Railway сам установит зависимости из `requirements.txt`

---

### Шаг 5 — Добавь переменные окружения

В Railway → твой проект → **Variables** → добавь:

| Переменная | Значение |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен от BotFather |
| `ANTHROPIC_API_KEY` | ключ от Anthropic |
| `GROUP_CHAT_IDS` | ID групп (см. ниже) |
| `MANAGER_IDS` | Telegram ID руководителя |
| `DB_PATH` | `/app/f2b_bot.db` |

**Как узнать ID группы:**
1. Добавь **@userinfobot** в нужную группу
2. Напиши `/start` — бот покажет ID группы (отрицательное число, например `-1001234567890`)

**Как узнать свой Telegram ID:**
1. Напиши **@userinfobot** в личку
2. Он покажет твой числовой ID

---

### Шаг 6 — Добавь бота в группы

1. Открой нужную группу в Telegram
2. Настройки → Участники → Добавить
3. Найди своего бота по username
4. Дай боту права **читать сообщения**
5. Готово!

## 💬 Как пользоваться

### Обращение к боту
Пиши в начале сообщения: **бот,** или **@имя_бота**

```
бот, пришли фото тунца
бот, какая цена на лосось МРМ?
бот, мои задачи на сегодня
бот, контакт Малахова
бот, составь вежливый отказ клиенту по срокам
```

### Команды
```
/задачи        — мои открытые задачи
/все_задачи    — все задачи команды
/просрочка     — просроченные задачи
/отчет         — недельный отчёт
/дебиторка     — срез по дебиторке
/фото [товар]  — найти фото товара
/прайс         — актуальный прайс
/контакт [имя] — найти контакт
/помощь        — все команды
```

### Автоматические напоминания
| Время | Что происходит |
|---|---|
| 09:00 | Утренняя сводка: задачи на день + просрочка |
| 10:00 | Напоминание по задачам на сегодня |
| 17:00 (пн-пт) | Напоминание Беляковой — реестр в склад |
| 18:00 | Запрос среза по дебиторке |

### Автоматическое создание задач
Бот читает **все сообщения руководителя** и автоматически извлекает задачи.

Например, руководитель пишет:
> "Карина, сверься с Ореховым по оплате до пятницы"

Бот создаёт задачу: **исполнитель** = Карина, **дедлайн** = пятница

### База фото и документов
Бот сохраняет **все фото и файлы** которые присылают в группах.
Чтобы фото находилось по запросу — добавь подпись при отправке:
> (отправляешь фото тунца с подписью "тунец саку ааа")

Потом любой менеджер пишет:
> "бот, пришли фото тунца" → бот находит и отправляет

## 🔧 Частые вопросы

**Бот не отвечает в группе?**
→ Проверь что бот добавлен в группу и имеет права читать сообщения

**Задачи не извлекаются автоматически?**
→ Проверь что `MANAGER_IDS` содержит правильный Telegram ID руководителя

**Как добавить несколько групп?**
→ В `GROUP_CHAT_IDS` перечисли через запятую: `-1001111111,-1002222222`

**Как обновить бота?**
→ Запушь изменения в GitHub, Railway передеплоит автоматически
